# Light color themes for Emacs and Vim

Light color themes for Emacs and Vim, easy on eyes with great readability.

- ivanb-theme.el
- ivanb.vim

Other files in this repository are surplus and are not required by color themes.

## Example

![Example](https://raw.githubusercontent.com/ivbit/ivanb-theme/main/example.png)

## Emacs commands:

**M-x** global-display-line-numbers-mode

**M-x** display-line-numbers-mode

**M-x** display-fill-column-indicator-mode

**M-x** hl-line-mode

**M-x** view-mode
